WP Site Portfolio plugin , build a portfolio for sites / templates you developed and display them in a responsive view: desktop, tablet, mobile.



== Description ==
WP Site Portfolio plugin will add to you wordpress admin a new post type and menu item that will give you the ability to showcase  your work to potential clients.
The plugin comes with shortcodes to show each website or theme you created in 3 view options: desktop, tablet and mobile.
All you need to do is create a new Site portfolio item, give it a title, add featured image and enter the website URL to create the preview.
the shortcode will add a beautyfull grid layout with default 3 items in a row, you can change the layout via the wp customizer.
You can customize the  size of each view option and add your logo to the toolbar in the settings page.
You can create categories and set each Site Portfilio item to a categiry.
You can customize the colors of the preview (toolbar bg, title color etc...) on the wp customizer.


== Installation ==
Install like any other plugin, directly from your plugins page.

1. Upload the plugin files to the `/wp-content/plugins/wp-site-portfolio` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Here's a link to the [plugin installation tutorial](https://youtu.be/tG8YEgQYG7g "Wp Site Portfolio Tutorial")


== Frequently Asked Questions ==
= How to show the portfolio on the website? =

Use the short code- [site_portfolio] in the text editor or your page builder.

= How to change the grid layout? =

Go to Appearance -> Costumize.
In the costumizer go to Wp Site Portfolio Settings.
Go to  Wp Site Portfolio Grid Settings and change the layout.

= How to change the Colors? =

Go to Appearance -> Costumize.
In the costumizer go to Wp Site Portfolio Settings.
Go to  Wp Site Portfolio Colors Settings and change the colorse.
